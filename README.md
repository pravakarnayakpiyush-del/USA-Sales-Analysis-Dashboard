## 📌 Project Overview
This project features a comprehensive **United States Sales Dashboard** designed to analyze and visualize sales performance, profitability, and customer behavior across different regions. It provides actionable insights into key business metrics by comparing Year-over-Year (YoY) data (2019 vs. 2020) and breaking down sales by various categories, segments, and shipping modes.

## 📊 Key Performance Indicators (KPIs)
At a glance, the dashboard tracks the following primary metrics:
* **Total Sales:** 0.34M
* **Total Profit:** 27.5K
* **Total Quantity Sold:** 5.2K
* **Average Delivery Days (D-days):** 4.04

## 🚀 Features & Visualizations
The dashboard includes dynamic filtering by region (Central, East, South, West) and features the following visualizations:

1.  **Trend Analysis:**
    * Monthly Sales by YoY (2019 vs 2020) - *Area Chart*
    * Monthly Profits by YoY (2019 vs 2020) - *Area Chart*
2.  **Categorical Breakdown:**
    * Sales by Category (Office Supplies, Furniture, Technology) - *Bar Chart*
    * Sales by Sub-Category (Chairs, Binders, Phones) - *Bar Chart*
3.  **Customer Insights:**
    * Sales by Segment (Consumer, Corporate, Home Office) - *Donut Chart*
    * Sales by Payment Mode (COD, Online, Cards) - *Donut Chart*
4.  **Logistics & Geography:**
    * Sales by Ship-Mode (Standard, Second Class, First Class, Same Day) - *Bar Chart*
    * Sales and Profit by State - *Geospatial Map Visualization*
